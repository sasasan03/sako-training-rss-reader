# sako-training-rss-reader

アプリ『RSSリーダーサンプルアプリ』

-__動作環境__:iOS 17.5以降
-__Swift_Version__:5.10
-__xcode_Version__:15.4

## 概要
- ログイン画面：Firebase(googleアカウントでのログイン)
- ニュースフィード選択画面(Yahoo!)
- ニュースフィード一覧画面(Yahoo!)/ニュースフィード詳細画面
- 設定画面/設定詳細画面

## 使用技術
- SwiftUI
- Firebase
- Yahoo_RSS(rss_end_point)
- xmlCoder 
# 外部ライブラリはSPMで導入
